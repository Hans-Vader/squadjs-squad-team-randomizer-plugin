# Squad Team Randomizer Plugin

# WIP

## Overview
- Randomizes both teams while ensuring squad members stay together.
- Triggered via the in-game admin chat command `!squadrandomize`.

## Requirements
- SquadJS server with RCON access.
- Admin permissions to run chat commands.

## Usage
1. Install the plugin alongside your other SquadJS plugins.
2. Configure the optional `command` setting if you need a custom trigger.
3. In admin chat, type `!squadrandomize` to shuffle teams and keep squads intact.
